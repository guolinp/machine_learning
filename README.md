## Tensorflow docker help
* Start docker
    ```
    cd docker
    ./start.sh
    ```

* Stop docker
    ```
    cd docker
    ./stop.sh
    ```

* Install packages into docker
    ```
    cd docker
    ./install_package.sh  # edit it if need
    ```

* Enter docker shell
    ```
    cd docker
    ./enter_shell.sh
    ```
